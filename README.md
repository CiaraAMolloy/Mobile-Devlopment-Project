instructions
Make flashcard sets:

1)In home click make Sets 

2)Choose a folder from the spinner

3)Give your set a unique name

4)click add

Make flashcards:

1)click make flashcards

2)select a set to add your flashcard to 

3)give your flashcard a term and definition
example term:1+1 definition:2 

4)click add

5)you should see the flashcard you created

Delete flashcards:

1)In home click Delete/view flashcards

2)select a set to delete cards from from the spinner

3)click the select button

4)you should now see the list of flashcards select one or multiple to delete

5)click delete

View cards in a set:

1)In home click Delete/view flashcards

2)select a set view

3)click the select button

4)you should now see the list of flashcards 

Play the flashcard game

1) In home click play flashcard game

2) click play game

3) select a set

4) choose a finish all cards or timed game

5) type the answers (no terms will appear if you do not make sets)

Share the results:

after you have played a game you can then click share result as text

Calender/creating events
1) In home click the calender button

2) click on a day to add an event (such as a test)

3) type in the name of your event

4) click add event

5) you will now see the days till that event
